# Teltonika Flespi Telegram Bot for FMC920 and FMB920

A production-ready no-webhook, no-domain, no-Docker Teltonika + Flespi + Telegram solution for FMC920/FMB920 GPS trackers. Teltonika sends telemetry to Flespi; a lightweight Python bot polls Flespi REST API and Telegram Bot API long polling to send Google Maps location, OpenStreetMap live tracking, ignition updates, towing alerts, daily idling reports, average trip time, and number of trips.
