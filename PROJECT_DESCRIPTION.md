# Teltonika Flespi Telegram Bot for FMC920 and FMB920

A ready-to-deploy Teltonika + Flespi + Telegram solution for FMC920 and FMB920 trackers. The Teltonika device sends GPS telemetry to a Flespi Teltonika channel using TCP or MQTT, and this bot reads normalized Flespi telemetry to send Telegram messages with Google Maps location, OpenStreetMap live tracking link, ignition status, towing alerts, and automatic daily reports at 12:00 AM for previous-day idling time, average trip time, and number of trips.

SEO keywords: Teltonika Telegram bot, FMB920 Telegram alerts, FMC920 Telegram tracking, Flespi Telegram integration, Teltonika Flespi MQTT, Teltonika TCP tracking, Google Maps Telegram GPS, OpenStreetMap live tracking, Teltonika towing alert, Teltonika idling report, fleet tracking Telegram bot.

End users can also change the location reporting interval and daily report time directly from Telegram using commands such as `/interval 5min`, `/interval 1h`, `/interval off`, and `/daily 00:00`.
