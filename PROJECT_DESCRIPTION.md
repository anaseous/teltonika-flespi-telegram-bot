# Teltonika Flespi Telegram Bot for FMC920 and FMB920

A production-grade Teltonika + Flespi + Telegram solution for FMC920/FMB920 GPS trackers. The tracker sends telemetry to Flespi; Flespi forwards normalized messages to the Telegram bot service; Telegram users receive Google Maps location, OpenStreetMap live tracking, ignition updates, towing alerts, daily idling reports, average trip time, and number of trips. The Telegram bot also uses Flespi REST API to answer `/status` and `/location` commands.
