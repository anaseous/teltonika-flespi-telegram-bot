# Project Description

Teltonika + Flespi + Telegram architecture for FMC920/FMB920 GPS trackers. Teltonika sends data to Flespi. Flespi parses telemetry, manages rules and intervals, sends Telegram messages, receives Telegram commands through the Flespi Telegram protocol, and updates Flespi configuration through the Flespi REST API. No Python, no Docker, no VPS, no Raspberry Pi, no domain, and no external server.

SEO keywords: Teltonika Telegram bot, FMC920 Telegram tracking, FMB920 Telegram alerts, Flespi Telegram integration, Flespi Telegram protocol, Teltonika Flespi REST API, Google Maps GPS Telegram, OpenStreetMap live tracking, Teltonika towing alert, Teltonika idling report.
