# Production Checklist

- Teltonika device sends data to Flespi successfully.
- Flespi device IMEI is correct.
- Flespi Telegram channel receives Telegram commands.
- Telegram bot token is stored only in Flespi.
- Telegram chat ID is verified.
- Location message template is tested.
- Towing alert is tested.
- Daily report is tested.
- Interval command is tested.
- Flespi token permissions are limited where possible.
- No external app, server, Docker, Raspberry Pi, webhook receiver, or public domain is used.
