# Teltonika FMC920/FMB920 Setup

## Flespi Server Settings

Use the domain and port generated by your Flespi Teltonika channel.

Example only:

```text
Domain: abc123.flespi.gw
Port: 21543
Protocol: TCP
```

## Recommended Teltonika Parameters

Enable these where available:

```text
GNSS location
Position speed
Ignition status
Movement status
Towing event
External voltage
Battery voltage
Trip odometer
```

## Recommended Sending Interval

Initial test values:

```text
Moving: every 30 seconds
On stop: every 300 seconds
```

The end user can later change reporting behavior through Telegram commands handled by Flespi.
